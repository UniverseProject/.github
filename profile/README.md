## 🙋‍♀️ KOTLIN VS JAVA

The java example:
```java
switch(days) {
  case Monday:
    // Your action for monday
    break;
  case Tuesday:
    // Your action for tuesday
    break;
  case Wednesday:
    // Your action for wednesday
    break;
  default:
    // Your default action
    break;
}
```
The kotlin example:
```kotlin
when(days) {
  Monday -> Your action for monday
  Tuesday -> Your action for tuesday
  Wednesday -> Your action for wednesday
  else -> // Your default action
}
```
